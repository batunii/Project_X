
@startuml
actor User
participant "Input Manager" as Input
participant "HUD Manager" as HUD
participant "Room Manager" as Room
participant "Security Manager" as SM
participant "Event Logger" as Logger

title Create an art gallery

== Basic Flow ==

User -> Input: clicks on HUD icon
Input -> Logger: log user event
Input -> HUD: requests HUD
User -> Input: clicks on create art gallery
Input -> Logger: log user event
Input -> HUD: validates and sends event information to HUD
Input -> Logger: log user event
HUD -> User: modal with options (gallery name, privacy type) is opened
User -> Input: Enters gallery's name and privacy type (public or private)
Input -> Logger: log user event
Input -> HUD: validates and sends event information to HUD
HUD -> Room: requests a gallery with user information
 
== Alternative Flow 1 – Private Gallery ==

Room -> SM: creates a gallery and requests a security layer.
SM -> Room: encrypts the gallery to make it private
Room -> User: notifies creation of private gallery.

== Alternative Flow 2 – Public Gallery ==

Room -> User: notifies creation of public gallery

@enduml
