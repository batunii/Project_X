@startuml
title Use Case — Create GenAI Art

actor User
participant "Controller/Input Manager" as Input
participant "HUD" as HUD
participant "Art Tools" as Tools
participant "Canvas" as Canvas
participant "Network Manager" as Net
participant "Event Logger" as Log

User -> Input : Opens HUD and taps "Create GenAI Art"
Input -> HUD : Prompt keyboard pops up
HUD -> Input : User enters prompt
Input --> HUD : Prompt processed when "Generate" clicked on
Log --> Log: Log: GenAI.GenerateClicked
HUD -> Tools : "Generate" button changes to "Generating" 
Log --> Log: Log: GenAI.RequestStarted
Tools -> Net : GenAI Model receives the prompt and starts generating the art
Net --> Tools : GenAI is ready
Log --> Log: Log: GenAI.ResultReady
Tools -> Canvas : GenAI art is attached to the canvas
Canvas --> Tools : Feedback received about ready image
Log --> Log: Log: GenAI.CanvasUpdated
Tools --> HUD : Art generation complete
HUD -> User : Displays generated artwork
Log --> Log: Log: GenAI.Displayed

alt Save and Publish
    User -> HUD : Taps "Save" then "Upload"
    Log --> Log: Log: GenAI.SaveRequested
    HUD -> Canvas : Exports current canvas image 
    Canvas --> HUD : Image is converted to an exportable format
    Log --> Log: Log: GenAI.ExportReady
    HUD -> Net : Uploads local file
    Net --> HUD : Gives object URL
    Log --> Log: Log: GenAI.UploadComplete
    HUD -> User : Art published successfully
    Log --> Log: Log: GenAI.Published
end
@enduml

@startuml
actor UserA
actor UserB

participant "Controller/Input Manager" as CIM
participant "HUD" as HUD
participant "Collaboration/Communication Manager" as CCM
participant "Network Manager" as NM
participant "Event Logger" as EL

== Proximity Voice Chat Flow ==
UserA -> CCM: Connects to local voice chat (initial volume=0)
UserA -> NM: Moves closer to UserB
NM -> CCM: Updates distance, adjusts UserA's voice volume for UserB
UserA -> CIM: Speaks (voice input)
CIM -> CCM: Forwards voice data (distance-based volume)
CCM -> NM: Broadcasts voice to UserB

alt Leaving the chat
UserA -> NM: Leaves UserB's range (volume decreases)
CCM -> EL: Log proximity/volume event
EL -> EL : Log Mute Action
end

== Public ChatBox Flow (Text) ==
UserA -> HUD: Opens HUD
HUD -> CCM: Selects public chat
CCM -> NM: Get history
NM -> CCM: Returns messages
CCM -> HUD: Shows history
UserA -> CIM: Types message, clicks send
CIM -> CCM: Forwards message
CCM -> NM: Broadcasts to users
CCM -> EL: Log message event
EL -> EL: Log Mute Action

== Private ChatBox Flow (Text) ==
UserA -> HUD: Selects UserB for private chat
UserA -> CIM: Points at UserB, selects chat option
CIM -> CCM: Initiate private chat
CCM -> NM: Opens private chat channel
NM -> UserB: Delivers private chat message
@enduml

actor User
participant "Controller/Input Manager" as Input
participant "HUD Manager" as HUD
participant "Canvas/Platform Manager" as Canvas
participant "Art Tools Manager" as Tools
participant "Network Manager" as Network
participant "Event Logger" as Logger

title Create 3D Art
User -> Input: clicks on a platform on the floor
Input -> Canvas: notify platform selection
Canvas -> HUD: confirm platform ready
Canvas -> Logger: log("PLATFORM_INITIALIZED", platformId)
Canvas -> HUD: zoom into platform view
HUD -> User: display empty surface

== Choosing a Shape ==
User -> HUD: chooses default shape (cube/sphere/cylinder/cone)
HUD -> Canvas: request shape creation
Canvas -> Tools: initialize default shape with properties
Tools -> Canvas: return shape object
Tools -> Logger: log("SHAPE_CREATED", shapeType, shapeId)
Canvas -> HUD: place shape at center of platform
HUD -> User: display shape on canvas

== Choosing an Art Tool ==
User -> Input: picks up an art tool
Input -> Tools: selects the art tool
Tools -> HUD: shows tool selection feedback
HUD -> User: tool selected confirmation

alt Select Platform from HUD
    User -> HUD: selects platform from HUD while in workspace
    HUD -> Canvas: request and load platform selection
    Canvas -> Network: fetch previously saved 3D art
    Network -> Canvas: return saved art data
    Canvas -> Logger: log("ART_RESTORED", platformId, objectCount)
    HUD -> User: show previously saved 3D art on platform surface
end

alt Change Art Tool
    User -> Input: switches to different art tool
    Input -> Tools: deactivate current tool, activate new tool
    Tools -> Logger: log("TOOL_CHANGED", newTool)
    Tools -> HUD: update tool selection indicator
    HUD -> User: show new tool active
end

      Network -> Event : Unsaved
      Network -> Room : Tool usage not saved
      Room -> HUD : Saving error
      HUD -> User : Display 'Error: Customisation not saved'
    end
  end
else User unauthorised
  Auth -> HUD : Send failed request
  HUD -> User : Display "Access not permitted"
